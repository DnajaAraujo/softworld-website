The Cursed Sword

Sobre o jogo

  • Gênero: Plataforma/Aventura
  • Plataforma: PC (Windows e Linux)
  • Modo de Interação: Single Player

História

O personagem principal retirou uma espada amaldiçoada e agora o mundo está entrando em ruínas, devido a isso, ele tem que levá-la para o monte sagrado antes que o mundo seja destruído.

Controles

  • Pular: seta para cima, space, botão direito do mouse
  • Andar: seta para direita/esquerda
  • Pulo Duplo: seta para cima (duplo clique), space (duplo clique), botão direito do mouse (duplo clique)
  • Usar Poder: tecla X

Requisitos mínimos para o PC

  • Memória RAM de 2GB
  • 1GB de Armazenamento Interno
